# vertex
Sitio de ventas, contactennos al grupo de discord

## Uso
Abre `index.html` en un navegador para ver el sitio estático. No se necesita servidor adicional.

## Funcionalidad
- Calculadora interactiva de Robux
- Enlaces directos a Discord y perfil de Roblox
- Secciones de precios, catálogo y FAQ
